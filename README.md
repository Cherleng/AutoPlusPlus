# AutoPlusPlus

English | [简体中文](./README-zh_CN.md)

Intelligent Car plate detection System based on OpenCV

## Compile

<details>
<summary><strong>Build</strong></summary>
<br />

1. Clone the repository:

```bash
git clone https://github.com/josedelinux/AutoPlusPlus.git
cd AutoPlusPlus
```

2. `make`

```bash
make
```

3. start the program

```
./dist/main/main
```

</details>
